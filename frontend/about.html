ConnExplorer

About

ConnExplore visualizes connectivity matrices (CM) within anatomical space. The left view shows the parcellation used to build the CM, while the right view displays connectivity for the selected region. Navigate by clicking the brain views or entering coordinates in the parameters panel. Choose parcellations and matrices from the dropdowns above the views.

How to Use ConnExplorer

1. Choose data

Select the parcellation and connectivity matrix from the dropdowns above the brain views.

2. Navigate anatomy

Click on the brain images to jump to a region, or type MNI coordinates in the Parameters panel.

3. Adjust visibility

Use Color Range to set the value scale and Threshold to hide low/high values.

4. Tune appearance

Select a colormap for each side to control visual contrast.

5. Explore matrix

Click any matrix cell to sync both views to that source/target; toggle labels or hemispheres as needed.

6. Share

Generate a link with the Share button to preserve the current state.

7. Color range

Define the scale for the colorbars.

8. Threshold

Set lower and upper bounds for values to be exculded.

Data

Parcellations

The available parcellation is the Automated Anatomical Labelling atlas, version 3.1 (AAL3, Rolls et al. 2020). It is a whole-brain parcellation with 166 labelled cortical and subcortical regions, available for free download.

Connectivity matrices

We provide the functional connectivity relevance matrix describing associations between AAL3 regions and alcohol use disorder, derived from “Relating functional connectivity and alcohol use disorder: A systematic review and derivation of relevance maps for regions and connections.” Published in OSF under CC-BY 4.0.

References
